# section01 버전관리의 본질

<br>

## git init

- pwd : 현재 위치를 알려준다
- mkdir : 폴더 생성
- cd : 체인지 디렉토리(이동하기)
- ls -al 파일 확인
- git init : 깃 start 방법 중 하나

<br>

## git add

- vim f1.txt
  - i 로 insert 모드, esc로 편집모드 종료
  - :wq를 적으면 저장(w) 후 나가기(q)
- cat f1.txt
  - 파일 안에 내용 보기
- git status
- git add f1.txt

<br>

## 버전 만들기

- 한번만 하면 되는 것 
  - git config --global user.name [user name]
  - git config --global user.email [user email]
- git commit
  - vim 화면 출력
  - :wq 엔터
- git log
